Trigger Import

The Trigger Import dialog allows you to import or export trigger entries as plain text for use with other programs. The file format consists of Tab Separated Values (TSV), one entry per line, separated by a CR (standard Macintosh line ending). Up to four fields can be imported but only the first field is required.

<IP_address> <trigger_type> <triggered_by> <trigger_duration>

Any omitted fields will use the default value specified in the Import Trigger dialog. The IP Address field can specify a single dotted quad, address range as A-B, or address and prefix length. The trigger types currently defined are: "trigger", "address", and "authorize".

Importing extends the current list of trigger entries based on the IP address specified. A new IP address will create a new entry, whereas a duplicate IP address or range will replace the previous entry.
